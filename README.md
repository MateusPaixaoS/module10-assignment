# Module 10 – Data Manipulation and Cleaning with Pandas

This repository contains my work for Module 10 of the course.

## Project Overview

This project cleans and transforms a messy **customer dataset** for UrbanStyle, a Florida-based retail clothing brand. The raw data contains missing values, duplicate records, inconsistent formatting, and mixed data types — all of which are addressed through a structured cleaning pipeline.

## Files Included

- `module10-assignment.py` – Main Python script covering data loading, cleaning, standardization, feature engineering, and a final business insights report.

## What I Practiced

- Detecting and handling missing values using strategies like median fill, forward fill, and mode imputation.
- Correcting data types for dates, currency strings, and numeric fields.
- Standardizing text data (names, categories, phone numbers) to a consistent format.
- Identifying and removing duplicate records.
- Engineering new features such as days since last purchase, average purchase value, and purchase frequency category.
- Generating business insights from cleaned data, including revenue by loyalty tier and category performance.
